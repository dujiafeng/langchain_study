# Deep Agents 学习路径

基于 LangChain Deep Agents 官方文档整理的 Jupyter Notebook 学习顺序。

---

## 第一阶段：入门（必读）

打下 Deep Agents 基础，理解 Harness 概念，跑通第一个 Agent。

| 顺序 | Notebook | 核心内容 | 预计时间 |
|:----:|----------|----------|:--------:|
| 1 | `01_核心概念/01_概述` | Harness 四大组件、框架定位、一句话创建 Agent | 15min |
| 2 | `01_核心概念/02_快速入门` | 完整 5 步搭 Agent：安装→API Key→搜索工具→创建→运行 | 20min |
| 3 | `01_核心概念/03_框架能力` | 执行环境、上下文管理、委派、引导 — 每个组件的详细说明 | 25min |
| 4 | `01_核心概念/04_对比` | Deep Agents vs Claude Agent SDK 异同对比 | 10min |

**产出：** 理解 Deep Agents 是什么、能做什么、和竞品比怎么样。

---

## 第二阶段：模型与配置（必读）

掌握模型选择和 Agent 定制。

| 顺序 | Notebook | 核心内容 |
|:----:|----------|----------|
| 5 | `02_模型与配置/01_模型` | provider:model 格式、ProviderProfile、动态切换 |
| 6 | `02_模型与配置/02_自定义` | 自定义 System Prompt、Tools、Middleware |
| 7 | `02_模型与配置/03_后端` | StateBackend / FilesystemBackend / StoreBackend / 自定义 |
| 8 | `02_模型与配置/04_配置档案` | HarnessProfile / ProviderProfile 注册与合并 |
| 9 | `02_模型与配置/05_MCP集成` | MCP 协议集成外部工具 |

**产出：** 能按需切换模型、自定义 Agent 行为、集成外部工具。

---

## 第三阶段：上下文与记忆（必读）

理解 Agent 的上下文管理，从输入控制到跨对话持久化。

| 顺序 | Notebook | 核心内容 |
|:----:|----------|----------|
| 10 | `03_上下文与记忆/01_上下文工程` | 五种上下文类型、System Prompt 组装顺序、压缩机制 |
| 11 | `03_上下文与记忆/02_长期记忆` | Store API、跨对话持久化、后台合并 |
| 12 | `03_上下文与记忆/03_技能` | 渐进式披露、SKILL.md、自定义技能 |
| 13 | `03_上下文与记忆/04_权限` | FilesystemPermission 规则、文件访问控制 |

**产出：** 掌握 Agent 的上下文生命周期和存储方案。

---

| 19 | `04_执行模式/06_Agent到Agent` | A2A 协议（了解即可） |
| 20 | `04_执行模式/07_通信协议` | ACP 协议（了解即可） |

**产出：** 掌握 Agent 的流式、子 Agent、异步和审批。
